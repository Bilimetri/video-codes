# Bilimetri — Animation Source Code

This repository contains the source code for the animated math videos published on the [Bilimetri YouTube channel](https://www.youtube.com/@Bilimetri). All animations are created using [Manim](https://www.manim.community/), a Python-based mathematical animation engine.
